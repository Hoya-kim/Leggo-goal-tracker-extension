# Leggo!🌟

골 트래커(Goal Tracker) 제작 프로젝트

- 사용자의 목표 달성에 동기부여를 줄 수 있는 프로젝트 제작

## Our Goal

- 사용자의 흥미를 유발할 유연한 UI/UX
- Vanilla JS, Clean code 작성
- Chrome Extension 제작 및 배포
